# team2-frontend

화면설계 기간에 사용할 Vue 3 기반 공통 프론트입니다.

## Stack

- Vue 3
- Vite
- Pinia
- Vue Router
- Tailwind CSS
- Axios
- Chart.js

## Run

```bash
npm install
npm run dev
```

## Directory

- `src/layouts`: 공통 레이아웃
- `src/views`: 서비스별 화면 초안
- `src/components`: 재사용 UI 조각
- `src/stores`: Pinia 상태
- `src/lib`: Axios 등 공통 유틸
- `src/data`: 화면설계용 mock 데이터
